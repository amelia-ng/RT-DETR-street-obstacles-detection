# Real-Time Street Obstacle Detection with RT-DETR

RT-DETR shows better results in detecting small and blurry obstacles on the road compared to YOLOv11n (Repo for YOLOv11n: https://github.com/amelia-ng/StrObsDet-yolov11n)

HuggingFace app: https://huggingface.co/spaces/ameliang/rt-detr-street-obstacle-detection
